# Hackathon Context

## NVIDIA Cosmos Cookoff

- Global virtual hackathon: Jan 29 - Mar 5, 2026
- Focus: Physical AI using NVIDIA Cosmos Reason 2
- Domains: Robotics, Autonomous Vehicles, Vision AI / Video Analytics
- Winners announced: week of March 16 (during GTC)

## Judging Criteria (4 dimensions)

| Criterion | What Judges Evaluate |
|---|---|
| Quality of Ideas | Compelling application of Cosmos Reason 2 for robotics/AV/vision AI |
| Technical Implementation | Clean code, reproducible, well-documented inputs/outputs/evaluation |
| Design | Well-thought-out architecture and UX |
| Impact | Real problem solved, advances Physical AI field |

## Submission Requirements

- Text description explaining features and functionality
- Demo video under 3 minutes
- Public code repository URL
- README with deployment/setup instructions

## Prizes

| Place | Prize |
|---|---|
| 1st | $3,000 + NVIDIA DGX Spark |
| 2nd | $2,000 + NVIDIA GeForce RTX 5090 |
| 3rd | $500 in Brev credits |

## Cosmos Reason 2 Model

- Architecture: Based on Qwen3-VL (multimodal visual + text)
- Sizes: 2B (24GB VRAM) and 8B (32GB VRAM)
- Context: 256K input tokens
- #1 open model on Physical AI Bench and Physical Reasoning leaderboards
- Capabilities: chain-of-thought physical reasoning, spatial-temporal understanding, 2D/3D localization, robot planning, video analytics
